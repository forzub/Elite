# 🌌 WORLD MODEL — Спецификация мира
## 1. Общие принципы
### Мир является:

стохастическим (содержит случайность),

активным (изменяется без участия игрока),

многоузловым (состоит из регионов / систем / миров),

графом зависимостей, проецированным на физическое пространство.

Игрок — не центр мира, а возмущение в нём.

События не прописываются вручную, а являются следствиями процессов.

## 2. Структура мира

Мир представлен как:

набор узлов (звёздные системы, миры, регионы),

связанных типизированными связями,

каждый узел имеет состояние и внутреннюю структуру.

## 3. Типы узлов
### 3.1 Метрополия

Центр управления, власти и инерции.

### 3.2 Колония

Зависимый узел, способный к развитию и потенциальной самостоятельности.

### 3.3 Экстрактивный узел

Ресурсный узел (шахты, астероиды, станции), не автономен.

### 3.4 Автохтонный узел

Местная цивилизация:

может быть мирной, агрессивной, фанатичной, светской, националистической,

может иметь нечеловеческую структуру (ульи, паразиты, симбионты и т.п.).

### 3.5 Пустой узел

Серые зоны:

пиратские базы,

изгнанники,

туристические странности,

неформальные сообщества.

## 4. Внутренняя структура узлов

Внутри узлов существуют подгруппы / фракции с различными интересами.

Состояния узла являются агрегатами внутренних противоречий.

## 5. Типы связей

Между узлами существуют связи:

Контроль (политическое подчинение)

Экономическая зависимость (ресурсы, энергия, миграция рабочей силы)

Военное давление

Культурное / идеологическое влияние

Информационные связи

## 6. Состояния узлов
### 6.1 Базовые

стабильность (стабильность / нестабильность / коллапс)

экономика (рост / стагнация / спад)

социальное состояние (довольство / напряжение / радикализация)

военное состояние (мир / конфликт / война)

биологическое состояние (здоровье / эпидемия / вымирание)

экологическое состояние (пригодность / деградация / катастрофа)

технологическое состояние (отсталость / развитие / прорыв)

### 6.2 Форс-мажорные (фазовые)

религиозный экстаз

золотой век

культурный ренессанс

апокалипсис

Форс-мажорные состояния временные и изменяют правила переходов.

## 7. Источники изменений
### 7.1 Социально-политические процессы

Внутренние конфликты, идеологии, управление.

### 7.2 Экономические процессы

Производство, торговля, зависимости.

### 7.3 Физико-природные процессы

Сведены к двум классам:

Медленные климатические тренды

Редкие катастрофические события

## 8. Уровень симуляции

Симулируются:

узлы (регионы),

ключевые агенты (лидеры, особые акторы, игрок).

Полной агентной симуляции нет.

## 9. Прозрачность мира

Мир полупрозрачный:

игрок видит последствия,

не видит формулы.

Информация может быть неполной, искажённой или запаздывающей.

## 10. Атмосфера

минимализм,

ощущение смысла,

не обязательно одиночество,

мир больше игрока.

## 11. Типы разума

Мир содержит агенты с различными логиками разума:

### 11.1 Антропный разум

Человекоподобная логика: цели, власть, рост, идентичность.

### 11.2 Коллективный разум

Роевые, ульевые и распределённые системы с приоритетом целостности структуры.

### 11.3 Экзистенциально-чуждый разум

Цивилизации и процессы, чьи цели:

не коррелируют с человеческими,

не направлены на власть, рост, экспансию или выживание в человеческом смысле,

могут быть связаны с поддержанием физических, энергетических, информационных или топологических состояний.

Их поведение:

наблюдаемо,

закономерно,

поддаётся анализу и моделированию,

но не интерпретируется через человеческие ценности.

Они могут:

перемещаться,

изменять среду,

строить или разрушать структуры,

взаимодействовать с физическим миром,

но не вступают в социальные, политические или экономические отношения с антропными цивилизациями.

### 11.4 Паразитический разум

Системы, использующие чужие структуры для размножения или экспансии.

## 12. Принцип ограниченной интерпретируемости

Не все процессы и агенты мира доступны для взаимодействия.

Однако они:

не являются случайными,

не являются декоративными,

имеют собственную внутреннюю логику и динамику.

Игрок может:

наблюдать их поведение,

строить гипотезы,

выявлять закономерности,

делать выводы и предсказания,

даже если не может с ними коммуницировать или влиять на них.

Это создаёт ощущение мира, который:

не вращается вокруг игрока,

но может быть понят в своих законах.
